# MealBridge

MealBridge is a simple web-based food donation platform built using HTML, CSS, and JavaScript.  
It helps reduce food waste by connecting donors and recipients through a browser-based system using LocalStorage.


## Features
Donate surplus food through form
Request available food items
View impact of donations
Track personal donation history
Awareness about food waste and hunger
Contact form for user queries
Data stored using LocalStorage
Responsive design for all devices

## Installation

1. Clone the repository
2. Install dependencies:
	- npm install

Running locally:
- npm start


## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- LocalStorage (Browser Storage)


## 📖 Usage

Navigate through the website using the Home page menu  
Go to Donate page to submit surplus food details  
Go to Request page to view and request available food  
Use My Donation page to track your donation history  
Check Impact page to view donation statistics  
Visit Education page for awareness about food waste  
Use Contact page to send queries or feedback  


## ⚙️ How to Run the Project

1. Download or clone the repository  
2. Open the project folder  
3. Open `index.html` in any web browser  
4. No installation or server required  


## 💾 Data Storage

- All data is stored in browser LocalStorage
- Includes:
  - Donation details  
  - Request details  
  - User activity data  


## 🎯 Objective

The main goal of MealBridge is to reduce food waste and connect people in need with surplus food through a simple and interactive web platform.
